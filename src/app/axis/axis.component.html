The Axises

Flexbox layouts are defined along two axes: the main axis and the cross axis.

  • Main Axis: This is the primary axis along which flex items are aligned. You can set it to be horizontal (default) or vertical using the flex-direction CSS property. Use a value of "row" for horizontal, or "column" for vertical.
  • Cross Axis: This is the perpendicular axis to the main axis.

    • The flex-direction property is applied at the container level.
