Alignment and Wrap

To align items along the Cross Axis, use the align-items CSS property. Some valid values for this are:

  • flex-start: Aligns items at the start of the Cross Axis.
  • flex-end: Aligns items at the end of the Cross Axis.
  • center: Aligns items at the center of the Cross Axis.
  • baseline: Aligns items so that their flex container baselines align.

    • This is not a complete list. For more information, check out this documentation.

    Also important is the flex-wrap property. This defaults to nowrap, which will try to place everything on the same line. But by setting this to wrap, the items will wrap when they need to.

    The flex-wrap and align-items properties are applied at the container level.
