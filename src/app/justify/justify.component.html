Justififcation

You can justify content using the CSS justify-content property. Some of the valid values of this are:

  • flex-start: This is the default and will align items with the start of the main axis.
  • flex-end: Aligns items with the end of the main axis.
  • center: (Not flex-center, just center.) This center aligns items.
  • space-between: Distributes items evenly.
  • space-around: Like space-between, but adds spacing around the edges.
  • space-evenly: Evenly distributes space in between all items.

    • This is not a complete list. More information can be found here.
