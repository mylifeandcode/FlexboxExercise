Flex

Using Flexbox, items can flex to fill the space. A few of the CSS properties that are useful for this are:

  • flex-grow: Sets the factor by which an element grows to fill the space.
  • flex-shrink: Sets the factor by which an element shrinks to fit the space.
  • flex-basis: Sets the initial main size of an element before space is distributed.
