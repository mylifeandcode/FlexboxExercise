This is all a lot to learn and can take time to get the hang of. For now, try to modify the styles for the items in the container below so that:

  • The first three items expand to fill the whole row. (Hint: These 3 items will need to grow.)
  • The second three have their own row and are centered vertically with space evenly distributed between them. (Hint: You can accomplish this at the container level.)
  • The last three have their own row, with the first item having a growth factor of 2. (Hint: Accomplish this at the item level.)

    • This will require creating new style classes, or applying inline styles to override the item class.
