The Pieces

When working with Flexbox, there are the following pieces:

  • Container - This is an HTML element with the CSS display: flex property applied to it.
  • Items - These are the items within the container, which will be displayed either in rows or columns (more on that later), and which you may want to resize proportional to the container and each other.

    • Here is a container with some items in it:

    1
    2
    3

    And here is the code for the container and elements.

    
<div style="display: flex; width: 600px; border-style: solid;">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>

    Additionally, the menu options shown above ("Introduction", "The Pieces", etc) are within a flex container. If you change the size of your browser window, you'll see the items shift and move accordingly.
